## FAQ
### 1. Why vote _abstain_?
* I'm not interested it it and want it to disappear from my search results
* Edit looks correct but verifying it takes too much time
* I just want to follow a discussion
* I don't know
* ...

In general, if you want people to vote _yes_ on your edits or as far as _approve_ them, **make [good edit notes](https://musicbrainz.org/doc/How_to_Write_Edit_Notes)**.

**Need votes** in a timely manner? Please post in the [Voting/Auto-editor Request Thread](https://community.metabrainz.org/t/21591/1).
Do you think I've **voted unfairly**? Please speak up **by leaving an edit note**. I'm always willing to rectify my votes or help you out.

### Where to find *barcode on digital releases*?
The [barcode](https://musicbrainz.org/doc/Barcode) field is supposed to store the EAN / UPC for digital releases. See [MBS-7419](https://tickets.metabrainz.org/browse/MBS-7419)
Those can often be found hidden somewhere on the site (check page source, URLs, etc.) or acquired via an API if available.

#### Spotify
* Use [the API](https://developer.spotify.com/documentation/web-api/) (usage of the web console is [not recommended](https://community.metabrainz.org/t/369137/5))

#### iTunes
* Ca. 2021 Apple started embedding data from the Apple Music API in the page source, there is a [userscript](#userscripts) to parse and display it
* Take a look at the thumbnail URL, sometimes it's in there.
* If you found the barcode there or somewhere else you can do a check with [their API](https://affiliate.itunes.apple.com/resources/documentation/itunes-store-web-service-search-api/)
* The [Enterprise Partner Feed](https://community.metabrainz.org/t/459687) is probably the best of all, if you have access to it.

#### Deezer
* Currently there is an unauthenticated API. For example: http://api.deezer.com/album/15242271
* Look at the page source and search for UPC

#### 7digital
* Use [the API](http://docs.7digital.com/) and try one of the ["free" oauth_consumer_key](https://musicbrainz.org/edit/66788791)

#### Bandcamp
* Look at the page source and search for UPC (warning: [might not be for the digital version](https://community.metabrainz.org/t/454077/11))
	* The [Bandcamp importer](https://github.com/murdos/musicbrainz-userscripts/blob/master/bandcamp_importer.user.js) displays it [on the page](https://community.metabrainz.org/t/454077/10).
* Open web console and inspect the `TralbumData` object
  * It's not yet documented so you need to figure out most of it on your own

#### HDtracks
* Use the API. The `productId` can be found in the URL of the release, e.g. `https://www.hdtracks.com/#/album/`**5e16dc6995ad4845eaed2d14**
	* Full release details: `https://hdtracks.azurewebsites.net/api/v1/album/<productId>`
	* Just title, description and UPC: `https://hdtracks.azurewebsites.net/api/v1/album/<productId>/meta`

#### Beatport
* Use [the API](https://api.beatport.com/v4/docs/)

#### Qobuz
* Use the API
